# Smart City Project

## 🚀 Overview
A full-stack Smart City solution designed to solve real-world urban challenges using modern technologies.

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- AI Modules: Traffic, Energy, Disaster Simulation

## ✨ Features
- Traffic prediction system
- Energy optimization dashboard
- Disaster simulation engine
- Urban planning tools
 
## 📂 Project Structure
- reactapp/ → Frontend
- springapp/ → Backend
